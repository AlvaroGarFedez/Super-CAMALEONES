Este proyecto es el original de la web Super-CAMALEONES (https://alvarogarciacei.es/), parte del Proyecto Kodama.

Incluye:
 - Página de entrada:
  -- Enlaces a la página principal y a la página sobre el Proyecto Kodama.
  -- Camaleón y título que cambian de color. 
 - Página principal:
  -- Título que cambia de color.
  -- Enlaces a la página de entrada y a la página sobre el Proyecto Kodama.
  -- Mapa de contenidos en forma de lista y como pines sobre la imagen de un camaleón, interrelacionados, que redirigen a los artículos principales de la web.
  -- Enlace a un último artículo con otros datos sobre los camaleones.
 - Artículos:
  -- Textos e imágenes que aparecen al hacer scroll.
 - Página sobre el Proyecto Kodama:
  -- Kodama que baja junto con el texto y "germina" al llegar al final.
 - Cabecera componentizada con botón back-link, común a todos los artículos y la página sobre el Proyecto Kodama.
 - Footer componentizado, común a toda la web.
 - Diseño responsive en el rango de 320px - infinito px.
